# 🎓 BIT Sindri — PE Result Portal

A fast, modern, and interactive GPA result portal built for the Production and Industrial Engineering department students of B.I.T. Sindri. This application allows students to instantly search, filter, and analyze their Semester 1 and Semester 2 results using a sleek, dark-mode-first interface.

## ✨ Features
* **Lightning Fast Search & Auto-Suggestions:** Find results instantly by Roll Number or Name with instant dropdown search recommendations showing chibi avatars and current academic levels.
* **Academic Duel Canvas:** Pit two student profiles side-by-side to compare SGPAs, overall CGPAs, rank standing differentials, and achievement quests.
* **Frosted Glass Scorecard PNG Download:** Export a high-resolution, glassmorphic card containing student GPA metrics, ranks, and dynamic digital signatures for offline storage.
* **Gender-Specific Chibi Anime Avatars:** Premium, hand-drawn vector chibi anime avatars mapped dynamically by student gender across all profile and list views.
* **Thematic SVG Class Icons:** Cohesive vector outline icons representing student class tiers (S-Class Sage, A-Class Specialist, B-Class Spellsword, C-Class Vanguard, D-Class Initiate, F-Class Challenger) replacing standard system emojis.
* **Class Analytics & CGPA Trend Charts:** View cohort averages, pass rates, SGPA grade distributions, and scrollable horizontal bar charts of individual student performance.
* **Dynamic Podium & Toppers Ledger:** Filter class toppers by cumulative Year 1 CGPA, Semester 1 SGPA, or Semester 2 SGPA, complete with visual podium rankings and detailed ledger lists.
* **Synthesized Retro SFX & Sound Control:** Interactive UI click blips, victory fanfares, and mute toggles built programmatically via the Web Audio API.
* **Fully Responsive:** Custom mobile swipers, horizontal scrolling cards, and flex-wrap grids for all viewports.

## 🛠️ Tech Stack
* **Frontend:** React (Hooks, useMemo, useEffect)
* **Animations:** GSAP (GreenSock Animation Platform)
* **Build Tool:** Vite
* **Styling:** CSS Variables (Custom Design System / index.css)
* **Deployment:** Vercel

## 🚀 Running Locally
To run this project on your local machine:

1. Navigate into the directory:
```bash
cd bit-sindri-production-portal
```

2. Install dependencies:
```bash
npm install
```

3. Start the local server:
```bash
npm run dev
```
